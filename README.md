# HOW TO USE THIS WEBSITE

Congratulations on your new portfolio! This website is built using only HTML, CSS, and JavaScript, making it very fast and easy to host on platforms like GitHub Pages.

### 1. Where to put files
- **Profile Photo:** Rename your photo to `profile.jpg` and place it in the `assets/` folder.
- **Resume:** Rename your resume to `Gaurav_Kolte_Resume.pdf` and place it in the `assets/` folder.

### 2. How to edit text
Open the following files in any text editor (like VS Code, Notepad++, or TextEdit):
- `index.html`: Homepage content, tagline, and intro.
- `about.html`: Detailed bio, education, and hobbies.
- `projects.html`: Your projects list and descriptions.
- `resume.html`: Resume page text.
- `style.css`: Colors, fonts, and spacing. Look for the `:root` section at the top to change primary colors.

### 3. How to update links
Search for the text `[ADD_YOUR_..._HERE]` in all HTML files and replace it with your actual email and social media profile URLs.

### 4. How to publish on GitHub Pages
1. Create a new repository on GitHub (e.g., `my-portfolio`).
2. Upload all files (`index.html`, `style.css`, `script.js`, `assets/`, etc.) to this repository.
3. Go to **Settings** > **Pages** in your GitHub repository.
4. Under "Build and deployment", select the **main** branch and **(root)** folder.
5. Click **Save**. Your website will be live at `https://your-username.github.io/my-portfolio/` in a few minutes!

### 5. Features included
- **Dark Mode:** A floating toggle button at the bottom right.
- **Responsive Design:** Works on mobile, tablet, and desktop.
- **Animations:** Subtle fade-in effects as you scroll.
- **Beginner Friendly:** Clean code with plenty of comments.
